Cómo generar el og-image.jpg:
1. Abrí este archivo en Chrome o Edge.
2. Usá las DevTools → Ctrl+Shift+I → seleccioná el elemento <body>.
3. Click derecho sobre el body → Capture node screenshot (o usá una extensión como "Full Page Screen Capture").
4. Guardá la imagen como og-image.jpg en la carpeta /public/ del proyecto.
5. Verificá que mida exactamente 1200 × 630 px.
